Setup
=====

Install nodejs: [http://nodejs.org/download/](http://nodejs.org/download/)

Install yo


    $> sudo npm install -g yo


Install angular generator (for development only)


    $> sudo npm install -g generator-angular


Run
===


    $> npm install
    $> bower install
    $> grunt serve
    

Test
====


    $> grunt test
